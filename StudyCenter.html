Connect Google Classroom

Google Classroom requires OAuth. Follow the steps below to sync your assignments.

Option 1 — Use Classroom directly

1
Go to classroom.google.com and sign in with your school account.
2
Click To-Do in the left panel to see all pending assignments.
3
Bookmark classroom.google.com/u/0/a/not-turned-in/all for the fastest view.

Option 2 — Manual sync (recommended)

1
Open Classroom in one tab and StudyCenter in another.
2
Each morning, copy new assignments here with + Add Assignment.
3
Use + Add Homework during the day for anything assigned verbally.
💡 The manual morning review forces you to consciously see what's coming — which actually improves memory and planning!

Get the most out of StudyCenter on your Chromebook.

📌
Install as App
Look for the install icon (monitor with +) in Chrome's address bar, or ⋮ → Cast, save → Install page as app.
🪟
Split-Screen
Alt + ] snaps StudyCenter right, Alt + [ snaps Classroom left. Copy assignments side by side!
⌨️
Shortcuts
Esc closes modals. Enter submits forms. Tab moves between fields quickly.
📅
Morning Routine
Open StudyCenter → Classroom → copy new assignments → check Dashboard. 5 minutes, no surprises ever.
⏱️
Use the Pomodoro
Start a 25-min focus session for each subject. When the timer rings, take a real break — the science backs it!
🔔
Notifications
Click 🔔 at the top to enable browser notifications. You'll get pinged when assignments or tests are due soon.
📝
Tests Section
Use the Tests page to add upcoming exams and quizzes. Add topics to study, then check them off as you prep!
🖨️
Print to PDF
Click 🖨️ at the top to open a print-ready assignment list. Save as PDF or print for when you're offline.
